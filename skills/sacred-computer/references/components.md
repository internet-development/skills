# SRCL Component Reference

Complete list of components available in [Sacred Computer](https://sacred.computer) (SRCL).

Each component consists of a `.tsx` file and a `.module.scss` file in the `components/` directory of [internet-development/www-sacred](https://github.com/internet-development/www-sacred).

## Subdirectories

The `components/` directory also contains these subdirectories with additional utilities:

| Directory | Contents |
|-----------|----------|
| `detectors/` | Viewport and feature detection utilities |
| `examples/` | Example compositions and layouts |
| `modals/` | Pre-built modal content components |
| `page/` | Page-level layout components |
| `svg/` | SVG icon components |
